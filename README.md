# Aomi Documentation

Documentation site for [Aomi](https://aomi.dev) — the blockchain harness for agentic AI.

Built on [Mintlify](https://mintlify.com).

## Development

```bash
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.

## Structure

- `getting-started/` — quickstart and end-user channel guides
- `guides/` — platform integration tutorials
- `concepts/` — architectural and conceptual documentation
- `reference/` — API, CLI, SDK, and runtime references
- `resources/` — changelog, FAQ
