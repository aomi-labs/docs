This guide covers setting up the web chat widget as a product surface for your Aomi App. The widget gives you a complete chat interface out of the box, built on shadcn components that you own and can edit freely.

## Prerequisites

- **Next.js 15** app with the App Router
- **Tailwind CSS** configured
- **shadcn/ui** initialized (`npx shadcn init` already run, `components.json` present)

Starting from scratch:

```bash
npx create-next-app@latest my-app --typescript --tailwind --app
cd my-app
npx shadcn init
```

## Install the Widget

```bash
npx shadcn add https://aomi.dev/r/aomi-frame.json
```

This installs the `AomiFrame` component, all sub-components, and every dependency in one step.

### What Gets Installed

**npm packages:**
- `@aomi-labs/react` — headless runtime, hooks, API client
- `@assistant-ui/react` — chat primitives

**Component files:**
```
your-project/
├── components/
│   ├── aomi-frame.tsx                    ← Main shell
│   ├── control-bar/
│   │   ├── index.tsx                     ← ControlBar
│   │   ├── model-select.tsx              ← Model dropdown
│   │   ├── app-select.tsx                ← App/agent dropdown
│   │   ├── api-key-input.tsx             ← API key dialog
│   │   ├── wallet-connect.tsx            ← Wallet button
│   │   └── network-select.tsx            ← Network dropdown
│   ├── assistant-ui/
│   │   ├── thread.tsx                    ← Chat thread
│   │   ├── thread-list.tsx               ← Thread list
│   │   ├── threadlist-sidebar.tsx        ← Sidebar wrapper
│   │   ├── tool-fallback.tsx             ← Tool display
│   │   ├── markdown-text.tsx             ← Markdown renderer
│   │   ├── tooltip-icon-button.tsx       ← Icon buttons
│   │   └── attachment.tsx                ← File attachments
│   ├── ui/                               ← shadcn primitives (button, sidebar, popover, etc.)
│   └── wallet-tx-handler.tsx             ← Wallet transaction handler
├── lib/
│   └── utils.ts                          ← cn() utility
└── package.json
```

### Registry Architecture

Components are pulled from three registries:

| Source | What | Why |
| --- | --- | --- |
| **aomi.dev/r** | Customized components (AomiFrame, Thread, ThreadList) | Aomi-specific features like wallet integration |
| **r.assistant-ui.com** | Unchanged upstream components (markdown, tooltips) | Receives upstream fixes |
| **ui.shadcn.com** | Primitives (Button, Sidebar, Dialog) | Standard UI building blocks |

Since components are copied into your project, you own the source code and can edit anything.

## Set Environment Variables

Create `.env.local`:

```bash
# Required: URL of the Aomi backend
NEXT_PUBLIC_BACKEND_URL=https://api.aomi.dev

# Required for wallet connect / social auth
NEXT_PUBLIC_PARA_API_KEY=your_para_api_key
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_walletconnect_project_id
```

## Add AomiFrame to a Page

```tsx
import { AomiFrame } from "@/components/aomi-frame";

export default function ChatPage() {
  return (
    <div style={{ height: "100vh" }}>
      <AomiFrame
        backendUrl={process.env.NEXT_PUBLIC_BACKEND_URL!}
        height="100%"
        width="100%"
      />
    </div>
  );
}
```

## Configure Your API Key

### Via the ControlBar UI

The widget includes a ControlBar with an API key input. Click the key icon, paste your API key, and select your app. The key is stored in `localStorage`.

### Programmatically

```tsx
import { useControl } from "@aomi-labs/react";
import { useEffect } from "react";

function ConfigureOnMount() {
  const { setState } = useControl();

  useEffect(() => {
    setState({ apiKey: "sk-your-api-key", app: "mycoindex" });
  }, []);

  return null;
}
```

## Run Your App

```bash
npm run dev
```

Open `http://localhost:3000`. You should see:

- A sidebar for thread management
- A message composer at the bottom
- A ControlBar with model and app selectors
- Streaming AI responses when you send a message

## Customizing the Layout

Hide controls you do not need:

```tsx
<AomiFrame.Root height="100%" backendUrl={process.env.NEXT_PUBLIC_BACKEND_URL!}>
  <AomiFrame.Header
    withControl={true}
    controlBarProps={{
      hideWallet: true,
      hideApiKey: true,
    }}
  />
  <AomiFrame.Composer />
</AomiFrame.Root>
```

## Namespace Configuration

Add a registry namespace to `components.json`:

```json
{
  "registries": {
    "aomi": "https://r.aomi.dev"
  }
}
```

Then install with `npx shadcn add @aomi/aomi-frame`.

## Updating Components

```bash
npx shadcn add https://aomi.dev/r/aomi-frame.json --overwrite
git diff components/
```
