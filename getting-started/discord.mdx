The Aomi Discord bot is currently in development. It will bring the same AI assistant capabilities to Discord servers: streaming responses, tool calling, wallet integration, and seamless onchain transactions.

## Status

**Coming soon.** Join our [Telegram](https://t.me/aomi_sendit_bot) or use the [Web App](https://chat.aomi.dev) in the meantime.

## Planned Features

- Slash commands for all Aomi operations
- Streaming AI responses in Discord threads
- Wallet connection via mini-app
- Tool calls with inline results
- Thread management for persistent sessions
