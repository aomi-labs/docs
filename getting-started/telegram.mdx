The Aomi Telegram bot lets you interact with Aomi directly in Telegram. Add [@aomi_sendit_bot](https://t.me/aomi_sendit_bot) to your chats and start using it immediately.

## Getting Started

Open Telegram, search for `@aomi_sendit_bot`, and start a chat. The bot responds with streaming messages, interactive menus, and full tool calling support.

## Slash Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and see the welcome menu |
| `/app` | Switch between Aomi Apps |
| `/wallet` | Connect or manage your wallet |
| `/model` | Switch AI models |
| `/apikey` | Set an API key for private Apps |
| `/settings` | Configure bot preferences |
| `/sessions` | View and manage chat sessions |
| `/network` | Select blockchain network |
| `/help` | Show available commands |

## Wallet

Connect a wallet through the Telegram mini-app at `/wallet`. You can:

- View wallet status and balances
- Approve transactions from within Telegram
- Sign typed data (EIP-712)
- Switch networks
- Disconnect when done

## DM vs Group Chat

The bot works in both direct messages and group chats. In groups, the bot responds when mentioned or when replying to its messages. Use `/help` in a group to see available commands.

## Chat Sessions

The bot supports multiple sessions. Switch between them with `/sessions` or start fresh conversations without losing history.
